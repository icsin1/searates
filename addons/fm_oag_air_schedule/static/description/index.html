Flight schedule integration

Below are the steps to generate & Configure - the OAG API Key for Sandbox and Production account.

  • Go to https://developers.oag.com and do Sign In / Sign Up.
  • Once you logged in go to the Products menu
  • On products screen, from the list of products click on "Flight Info Trial" product for sandbox-API or click on "Flight Info" product for production-API.
    Description and limitations of the product is also mentioned here.
  • On the next screen, write name of you subscription (anything For e.g. "TEST" or "Flight Info - V2" and hit "Subscribe" button
  • Once you hit "Subscribe" button, it will redirect you to the User Profile screen. Where you will be able to see the subscribed information with API key informations. Hit "Show" to view and then you can copy the API key. you can regenerate API key from here once it is expired.
  • Configuration Screen: Paste you API key here and hit "Save".
